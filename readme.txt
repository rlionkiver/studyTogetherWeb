README - Study Together Website

Deskripsi Proyek
Website ini dibuat sebagai tugas pertama yang berfokus pada platform belajar bersama. Platform ini menyediakan berbagai fitur seperti grup belajar, alat bantu belajar, dan perencanaan studi untuk mendukung kolaborasi antar pengguna.

Fitur Utama
1. Navbar (Navigasi)
   - Logo situs
   - Menu navigasi: Study Groups, Tools, Profile
   - Tombol Sign In & Sign Up
   - Responsif dengan burger menu (untuk pengembangan ke depan)

2. First Section (Pengantar)
   - Slogan: "Study Together, Achieve Together"
   - Penjelasan singkat tentang kolaborasi belajar
   - Ilustrasi gambar

3. Study Groups Section
   - Menampilkan daftar grup belajar aktif berdasarkan kategori (Matematika, Sains, Bahasa, dll.)
   - Informasi jumlah peserta dan sesi belajar berikutnya
   - Tombol untuk bergabung dengan grup belajar

4. Study Tools Section
   - Alat bantu belajar seperti:
     - Virtual Study Room
     - Shared Documents
     - Study Planner
   - Setiap alat dilengkapi tombol aksi (Enter Room, Open Docs, Plan Now)

5. Footer
   - Informasi tentang pembuat proyek
   - Penjelasan singkat mengenai website

- **CSS**: Styling dan tata letak

Catatan Tambahan
- Untuk pengembangan ke depan, dapat ditambahkan fitur responsif pada navbar menggunakan JavaScript.
- Perbaikan pada tampilan agar lebih interaktif dengan efek hover dan animasi CSS.
